Beeferino Vokabelverwaltung – Komplettpaket
=============================================

Dateien:
- vokabeln.html   → Hauptseite (öffnet im Browser)
- vokabeln.css    → Layout & Design (Desktop + Mobile + Darkmode + A–Z Overlay)
- vokabeln.js     → App-Logik (Desktop-Tabelle, Mobile-Karten, Modal, Sortierung, GitHub-Sync)
- README.txt      → Diese Datei

Installation & Nutzung:
1) Lege alle Dateien im selben Verzeichnis ab.
2) Öffne vokabeln.html im Browser (Doppelklick).
3) Desktop: Tabelle mit Suche, Kategorie-Filter, Sortierung, A–Z-Overlay (Button unten rechts).
4) Mobile: Kartenlayout; Floating-Buttons (➕ und A–Z). Modal öffnet ebenfalls zum Bearbeiten.
5) Darkmode umschalten mit dem runden Button oben rechts (🌙 / ☀️).
6) GitHub-Sync: Beim ersten Upload Token (repo-Scope) eingeben; wird lokal gespeichert.

Hinweise:
- Kategorie-Farben bleiben in Selects sichtbar (auch im Darkmode).
- "Erstellt am" wird im Modal als Read-Only angezeigt.
- Verwechslungsgefahr-Schalter: Grün = an, Grau = aus.
- A–Z-Button zeigt den aktiven Buchstaben an; "Alle" setzt zurück auf A–Z.
